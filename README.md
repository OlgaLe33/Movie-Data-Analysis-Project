# Movie-Data-Analysis-Project
Analysis of movies from 1960 to 2000 using data from Wikipedia. The project includes processing JSON files, cleaning the data, visualizing top genres and actors, and analyzing their popularity trends over time.
## Overview

The project allows you to:
- Load and combine JSON files from different decades.
- Clean data and keep only complete records.
- Explore the top genres and actors by number of movies.
- Visualize the popularity trends of top genres and actors over time.
- Evaluate the proportion of movies for actors in top genres versus other genres.

## Technologies

- Python 3
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Requests
- Google Colab (for downloading files)

## Data

Data is sourced from the [Wikipedia Movie Data repository](https://github.com/prust/wikipedia-movie-data).  
Files used:
- movies-1960s.json
- movies-1970s.json
- movies-1980s.json
- movies-1990s.json
- movies-2000s.json

## Results
- Top 10 genres and their visualizations (bar and pie charts)
- Popularity trends of top 3 genres over time
- Top 10 actors and their popularity trends
- Visualization of the proportion of movies actors appear in top genres versus other genres
